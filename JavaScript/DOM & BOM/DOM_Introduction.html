DOM full form is Document Object Model hamara jitna bhi HTML ka code hota hai usse ham ek JavaScript object me convert kar lete hai jise document kaha jaata hai. or usse hee DOM yani ki Document Object Model kaha jaata hai. DOM hai wo ek Tree like structure hota hai. to jo hamara tag hote hai usme to characters hee hote hai. jesi ki html me h,t,m,l to jo wo characters hote hai wo convert ho jaate hai Tags me. lekin hame kese pata lage ga ki wo characters Tag hai to jo characters hai uske aage or piche yadi < or > hai yani ko wo ek tag hai to hamare characters convert ho jaaye ge Tags me. Fir wo Tags hai wo convert ho jaaate hai Tokens me jo ki karta hai hamara tokenizer. Fir us Tokens ko convert kiya jaata hai nodes me. or last me ussi nodes me hamara DOM ready ho jaata hai. DOM ka use JavaScript ki help se HTML Element add , remove, update ya read karne ke liye hota hai. DOM ka use JavaScript ki help se CSS ko bhi apply karne ke liye hota hai. dom ka use javascript ke help se html me 4 type ke task perform karne ke liye hota hai. 1) GET (Fetch) 2) SET (Update) 3) Add HTML Element (Add New Tag) 4) Delete HTML Element (Remove Old Tag)

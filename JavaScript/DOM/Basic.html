Sab pehle hame 3 topics ke baaare me idea hona chahiye. 1. window | 2. DOM | 3. BOM What is window? window hai wo ek global object hai. window object hai wo hamare browser window represent karta hai. jo window object hota hai usse browser create karta hai. window object me hee hamare DOM , BOM and JavaScript ke jo core concept hai to hote hai. global object yani ki jise ham kahi pe bhi use kar sakte hai. jese ki ham console.log() karte hai to ham use window.console.log() bhi likh sakte hai. yani ki window object ka use karke ham apne browser ko control kar sakte hai. What is DOM? DOM full form is Document Object Model hamara jitna bhi HTML ka code hota hai usse ham ek JavaScript object me convert kar lete hai jise document kaha jaata hai. or usse hee DOM yani ki Document Object Model kaha jaata hai. What is BOM? BOM full form is Browser Object Model. BOM ka use ham tab karte hai jab hame page ke contet ko chodkar koi or task karna ho like(location,history) jise ham browser ke saath communicate kar paaye. yani ki jab hame apne browser se communicate karna hai using JavaScript jo ki ham apne content ke alva kuch or task karna chalh te hai tab ham BOM ka use karte hai. it's allows JavaScript to talk to browser about metters other than content of page.
